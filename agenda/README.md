# Agenda Eventi 

Applicazione Python per la gestione di un'agenda personale
con persistenza dei dati su file di testo.

## Funzionalità
- Aggiunta di eventi con titolo, data, ora e descrizione
- Visualizzazione eventi per data specifica
- Visualizzazione dei prossimi N eventi in ordine cronologico
- Salvataggio e caricamento eventi da file

## Tecnologie utilizzate
- Python 3
- OOP — classi `Evento` e `Agenda`
- `datetime` — ordinamento cronologico eventi
- File I/O — persistenza dati su file .txt

## Come avviare il progetto

1. Clona il repository
2. Avvia il programma (nessuna dipendenza esterna):
   python agenda.py
