# Registro Studenti 

Applicazione Python per la gestione delle presenze scolastiche
con persistenza su file di testo.

## Funzionalità
- Registrazione presenza o assenza con data automatica
- Visualizzazione storico completo
- Conteggio totale presenze per studente
- Conteggio totale assenze per studente

## Tecnologie utilizzate
- Python 3
- `datetime` — data automatica ad ogni registrazione
- File I/O — persistenza dati su file .txt
- Gestione eccezioni con `try/except`

## Come avviare il progetto

1. Clona il repository
2. Avvia il programma (nessuna dipendenza esterna):
   python registro_studenti.py

## Esempio di output

2026-03-25: Studente: mario rossi è presente
2026-03-25: Studente: sara bianchi è assente
